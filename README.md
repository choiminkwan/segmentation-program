단축키 
	•	[ / ] : 브러시 두께 -/+ (1–30)
	•	E : Fill ↔ Erase 토글
	•	P : Polygon 탭으로 전환
	•	B : Pen 탭으로 전환
	•	Enter/더블클릭 : 폴리곤 완료(채우기/지우기 적용)
	•	Esc : 폴리곤 취소
	•	Backspace : 폴리곤 마지막 점 삭제
	•	Ctrl+Z : Undo


####
문제 사항
undo 개별 스택을 가지고 있지 않음 

원본과 마스크를 순서에 따른 매핑 -> 버그가 생긴다면 원본 파일명과 마스크 파일명을 보고 매핑 방식을 바꿔야 할 수도 있음
